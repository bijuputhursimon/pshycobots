📖 Psychology Bot Swarm - User Guide
🔑 HOW TO GET YOUR GEMINI API KEY
  1. Go to: https://aistudio.google.com/app/api-keys
  2. Click the blue "Get API key" button
  3. Select "Create API key in new project"
  4. Name your project (e.g., "Psychology Bot Swarm")
  5. Select "Gemini API" from the dropdown
  6. Click "Create Key"
  7. Copy the generated API key
  8. Paste it in the "Gemini API Key" field in Settings
  9. Click "Save Key" to encrypt and store locally
⚠️ SECURITY NOTE: Never share your API key. It's encrypted and stored locally on your device only.
🎯 GETTING STARTED
  • Set your Gemini API key in Settings (see above)
  • Create bot personas with distinct personalities
  • Form groups by combining personas
  • Chat with multiple personas simultaneously
🎭 PERSONAS
  • Each persona has unique system prompt and traits
  • Customize how they think, respond, and communicate
  • Edit or delete personas anytime
  • Click ✎ to edit, ✕ to delete
💬 GROUPS
  • Combine multiple personas in one conversation
  • All personas respond to your messages
  • Messages persist and personas remember context
  • Organize your conversations by topic or scenario
🎓 MENTOR BOT
  • Click "💡 Coach Me" button after any bot response
  • Or open Mentor modal and ask coaching questions
  • Get strategic guidance + response templates
  • Persona-aware coaching tailored to the bot
💾 DATA & PRIVACY
  • All data (personas, groups, messages) stored locally
  • API key is Base64 encrypted for security
  • Export your data anytime as JSON backup
  • Clear all data with one click (Settings → Clear All Data)
🚀 ADVANCED TIPS
  • Use personas for different perspectives in meetings
  • Mentor helps improve communication skills
  • Create groups for: sales scenarios, interviews, decisions
  • Edit persona prompts to fine-tune behavior
  • Export conversations for record-keeping
